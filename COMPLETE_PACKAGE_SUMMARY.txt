✨ AUTO-SAVE SYSTEM EXPLAINED
================================================================================

Every 10 Seconds (Automatic):
- Game data saved to browser localStorage
- Key: "scholasticAutoSave"
- Contains: competition name, team names, all scores, current round
- Survives: browser close, page refresh, computer restart

On Demand (Manual):
- Click "Save Game" button
- Saves to "Previous Games" list
- Key: "scholasticGames"
- Multiple saves can exist
- Can load any saved game anytime

Recovery (If Browser Crashes):
- App detects auto-save on startup
- Shows "Restore from Auto-Save" button
- Click to recover all data
- Never lose progress
